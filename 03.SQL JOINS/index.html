SQL JOIN


What is SQL JOIN?

A JOIN clause is used to combine rows from two or more tables, based on a related column between them.

Comman Example Query:

CREATE TABLE student (
     student_id INT PRIMARY KEY,
     name VARCHAR(50)
);
CREATE TABLE course (
     student_id INT UNIQUE,
     course VARCHAR(50)
);

INSERT INTO student(student_id,name)
VALUES

(102,'bob'),
(103,'casey');

INSERT INTO course(student_id,course)
VALUES
(102,'english'),
(105,'math'),
(103,'science'),
(107,'computer');

select * from student;
select * from course;

Note : It is not necessary for tables to be connected/relationship(1-1/1-M/M-M) through a primary key - foreign key relationship to perform SQL JOIN operations.



Different Types of SQL JOINS


1]INNER JOIN/JOIN :

Returns records that have matching values in both tables.
Syntax :SELECT column(s) FROM table1
              INNER JOIN table2
              ON table1.colname=table2.colname;

Ex1
select * from student as s
inner join course as c
on s.student_id = c.student_id;

Result
student_id name student_id course
102 bob 102 english
103 casey 103 science


2]LEFT JOIN/LEFT OUTER JOIN :

Returns all records from the left table, and the matched records from the right table.
Syntax :SELECT column(s) FROM table1
              LEFT JOIN table2
              ON table1.colname=table2.colname;

Ex2
select * from student as s
left join course as c
on s.student_id = c.student_id;

Result
student_id name student_id course
101 adam null null
102 bob 102 english
103 casey 103 science


3]RIGHT JOIN/RIGHT OUTER JOIN :

Returns all records from the right table, and the matched records from the left table.
Syntax :SELECT column(s) FROM table1
              RIGHT JOIN table2
              ON table1.colname=table2.colname;

Ex3
select * from student as s
right join course as c
on s.student_id = c.student_id;

Result
student_id name student_id course
102 bob 102 english
103 casey 103 science
null null 105 math
null null 107 computer


4]FULL JOIN/FULL OUTER JOIN :

Returns all records when there is a match in either left or right table.
Syntax :SELECT column(s) FROM table1
              LEFT JOIN table2
              ON table1.colname=table2.colname
              UNION
             SELECT column(s) FROM table1
              RIGHT JOIN table2
              ON table1.colname=table2.colname;

Note :In MYSQL ,there is no command called FULL JOIN ,so use UNION instead of them.

Ex4
select * from student as s
left join course as c
on s.student_id = c.student_id
union
select * from student as s
right join course as c
on s.student_id = c.student_id;

Result
student_id name student_id course
101 adam null null
102 bob 102 english
103 casey 103 science
null null 105 math
null null 107 computer

5]LEFT EXCLUSIVE JOIN :

Returns all records from the left table, and remove the matched records from the right table.
Syntax :SELECT column(s) FROM table1
              LEFT JOIN table2
              ON table1.colname=table2.colname
              WHERE table2.colname IS NULL;

Ex5
select * from student as s
left join course as c
on s.student_id = c.student_id;
where c.student_id is null;

Result
student_id name student_id course
101 adam null null


6]RIGHT EXCLUSIVE JOIN :

Returns all records from the right table, and remove the matched records from the left table.
Syntax :SELECT column(s) FROM table1
              RIGHT JOIN table2
              ON table1.colname=table2.colname
              WHERE table1.colname IS NULL;

Ex6
select * from student as s
right join course as c
on s.student_id = c.student_id;
where s.student_id is null;

Result
student_id name student_id course
null null 105 math
null null 107 computer


7]FULL EXCLUSIVE JOIN :

Returns all records from left and right table but remove the matched records from the left and right table.
Syntax :SELECT column(s) FROM table1
              LEFT JOIN table2
              ON table1.colname=table2.colname
              WHERE table2.colname IS NULL
              UNION
             SELECT column(s) FROM table1
              RIGHT JOIN table2
              ON table1.colname=table2.colname
              WHERE table1.colname IS NULL;

Ex7
select * from student as s
left join course as c
on s.student_id = c.student_id
where c.student_id is null
union
select * from student as s
right join course as c
on s.student_id = c.student_id
where s.student_id is null;

Result
student_id name student_id course
101 adam null null
null null 105 math
null null 107 computer


8]SELF JOIN :

It is a regular join,but the table is joined with itself.
Syntax :SELECT column(s) FROM table as a
              JOIN table as b
              ON a.colname=b.colname;

Ex8
